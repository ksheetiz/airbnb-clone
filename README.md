
# Airbnb Clone



## Run Locally

Clone the project

```bash
  git clone https://github.com/ksheetiz/airbnb-clone.git
```

Go to the project directory

```bash
  cd airbnb-clone
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn run dev
```
